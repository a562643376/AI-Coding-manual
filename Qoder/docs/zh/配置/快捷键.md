# 快捷键

Qoder 通过快捷键帮助你提升工作效率，涵盖行间会话、Agentic Chat、diff 视图导航等功能。

以下是按功能分组的默认快捷键完整列表。

<div id="general">
  ## **常用**
</div>

| **命令**      | **macOS** | **Windows**    |
| :---------- | :-------- | :------------- |
| 打开/关闭 聊天    | `⌘ L`     | `Ctrl L`       |
| 打开通用设置      | `⌘ ,`     | `Ctrl ,`       |
| 打开 Qoder 设置 | `⌘ ⇧ ,`   | `Ctrl Shift ,` |
| 打开命令面板      | `⌘ ⇧ P`   | `Ctrl Shift P` |

<div id="chat">
  ## **聊天**
</div>

| **命令**    | **macOS** | **Windows**      |
| :-------- | :-------- | :--------------- |
| 发送消息      | `⏎`       | `⏎`              |
| 在输入框中插入新行 | `⇧ ⏎`     | `Shift ⏎`        |
| 接受所有建议的更改 | `⌘ ⏎`     | `Ctrl ⏎`         |
| 拒绝所有建议的更改 | `⌘ ⌫`     | `Ctrl Backspace` |

<div id="inline-chat">
  ## **行间会话**
</div>

| **命令** | **macOS** | **Windows** |
| :----- | :-------- | :---------- |
| 打开行间会话 | `⌘ I`     | `Ctrl I`    |
| 发送到会话  | `⌘ L`     | `Ctrl L`    |

<div id="code-selection-and-context">
  ## **代码选中与上下文选择**
</div>

| **命令**        | **macOS** | **Windows** |
| :------------ | :-------- | :---------- |
| 将上下文添加到聊天     | `#`       | `#`         |
| 调用快捷命令        | `/`       | `/`         |
| 将当前选中代码块添加到聊天 | `⌘ L`     | `Ctrl L`    |

<div id="diff-view">
  ## **Diff 视图**
</div>

| **命令**     | **macOS** | **Windows**            |
| :--------- | :-------- | :--------------------- |
| 接受单个更改     | `⌘ ⇧ ⏎`   | `Ctrl Shift ⏎`         |
| 拒绝单个更改     | `⌘ ⇧ ⌫`   | `Ctrl Shift Backspace` |
| 接受文件中的所有更改 | `⌘ ⏎`     | `Ctrl ⏎`               |
| 拒绝文件中的所有更改 | `⌘ ⌫`     | `Ctrl Backspace`       |
| 跳转到下一个文件   | `⌘ ⇧ [`   | `Ctrl Shift [`         |
| 跳转到上一个文件   | `⌘ ⇧ ]`   | `Ctrl Shift ]`         |
| 跳转到下一个更改   | `⌥ J`     | `Alt J`                |
| 跳转到上一个更改   | `⌥ K`     | `Alt K`                |

<div id="quest-mode">
  ## **Quest Mode**
</div>

| **命令**        | **macOS** | **Windows**  |
| :------------ | :-------- | :----------- |
| 打开 Quest 面板   | `⌘ E`     | `Ctrl E`     |
| 打开全局 Quest 窗口 | `⌘ ⌥ E`   | `Ctrl Alt E` |
