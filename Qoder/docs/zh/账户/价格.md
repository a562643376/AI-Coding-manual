# 价格

Qoder 为个人用户提供 Free、Pro、Pro+ 和 Ultra 四种方案。如需团队协作，请参阅 [Teams 价格](https://docs.qoder.com/zh/account/teams/teams-pricing)。

<div id="which-plan-should-i-choose">
  ## 如何选择方案？
</div>

* **Pro**：适合日常开发，主要使用代码补全和轻量 Agent 辅助。
* **Pro+**：适合重度依赖 Agent 自主编码的开发者。
* **Ultra**：适合频繁使用 Quest 等长周期任务的用户，提供充足的资源额度。

<div id="checking-your-credits-usage">
  ## **查看 Credits 用量**
</div>

前往 **Settings >  Usage** 页面，即可查看当前订阅周期的 Credits 使用情况。不同功能消耗 Credits 的速率各不相同，详见 [Credits](https://docs.qoder.com/zh/Credits)。

<div id="what-happens-when-you-run-out-of-credits">
  ## **Credits 用尽后会怎样？**
</div>

Credits 即将耗尽或已耗尽时，您会在用量页面和 Qoder 客户端中收到提示。耗尽后系统会自动切换至基础模型继续服务（基础模型有每日使用上限，达到上限需次日恢复）。

<div id="credit-pack">
  ## **个人资源包**
</div>

Pro、Pro+ 和 Ultra 订阅用户可随时增购资源包来补充 Credits。当方案额度不够用时，资源包是便捷的补充方式。

### 价格与规格

| 项目   | 说明                                                                                        |
| :--- | :---------------------------------------------------------------------------------------- |
| 付费模式 | 预付费，单次购买，可多次叠加                                                                            |
| 单价   | 0.01 USD / Credit（[限时五折](https://docs.qoder.com/zh/events/discount)，原价 0.02 USD / Credit） |
| 起购量  | 1,000 Credits 起购                                                                          |
| 购买步长 | 以 1,000 Credits 递增                                                                        |
| 有效期  | 自购买日起 **1 个月**，到期清零                                                                       |
| 退款   | **不支持退款**                                                                                 |

<Info>
  可以同时持有多个资源包，每个资源包的有效期独立计算。
</Info>

### 购买步骤

1. 登录 [Qoder 网站](https://qoder.com/account/usage)，前往 **Settings >  Usage** 页面。
2. 在 **Get More Credits** 卡片中点击购买资源包。
3. 选择所需的 Credits 数量（最低 1,000，以 1,000 递增）。
4. 确认订单（含折扣明细）并完成支付。

<Tip>
  限时优惠：资源包和订阅升级均可享受五折优惠！
</Tip>

### 查看资源包

支付成功后，前往 **Usage** 页面的 **Add-on Credits** 卡片即可查看资源包的生效状态。Credits 获取记录中会新增一条"**资源包购买（Credit Purchase）**"的记录。

### 消耗优先级

多种资源来源（Plan Credits 及 Add-on Credits）统一按到期时间排序，最先到期者优先消耗。

<div id="pro-trial-rules">
  ## **Pro 试用规则**
</div>

首次登录 Qoder 客户端的新用户可获得 **14 天免费 Pro 试用**（需使用最新版本，且通过非虚拟机方式登录使用），附赠 300 Credits，体验 Pro 方案的全部功能。每位用户限享一次，创建多个试用账户将导致账号被冻结。

<div id="upgrading-your-subscription-plan">
  ## **升级方案**
</div>

可以随时升级到更高方案。升级时，当前未使用的 Credits 会自动结转为附加包，保留原有到期日，不会浪费。

* **从 Pro Trial 升级**：未用完的赠送 Credits 自动转为附加包，保留原到期日。
* **从 Pro 升级到 Pro+**：Pro 剩余的 Credits 自动转为附加包，保留原到期日；原 Pro 方案立即失效，新 Pro+ 周期随即开始。

<Info>
  升级不会浪费已有的 Credits，放心操作。
</Info>

<div id="downgrading-your-subscription-plan">
  ## **降级与取消**
</div>

可以随时取消订阅。取消后，当前方案在本计费周期结束前仍保持有效，之后账户自动降级为 Free。降级后，未使用的方案 Credits 将失效且不可转移。

暂不支持在订阅周期内直接降级到较低方案。

<div id="advanced-features">
  ## **高级功能**
</div>

付费方案相比 Free 额外提供以下功能：

* **Quest Mode**：面向复杂、长周期开发任务的 AI 辅助编程。详见 [Quest Mode](https://docs.qoder.com/zh/user-guide/quest-mode)。
* **Repo Wiki**：自动生成项目文档并持续追踪代码变更。详见 [Repo Wiki](https://docs.qoder.com/zh/user-guide/repo-wiki)。
