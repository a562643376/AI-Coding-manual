# 自定义模型

Qoder 支持通过 API 密钥接入供应商的 Coding Plan，以使用第三方模型资源。

<Note>
  **适用版本**：Pro Trial / Pro / Pro+ / Ultra

  **支持的供应商**：阿里云百炼、智谱、Kimi、MiniMax
</Note>

<Warning>
  **特别说明**：Repo Wiki 使用固定模型，采用独立计费方式。生成时会有提示通知您额外的 Credits 消耗。
</Warning>

## 添加自定义模型

**1：进入 Qoder 设置**

点击 IDE 左上角 Qoder，选择设置 -> Qoder 设置，打开 Qoder 设置面板

<img src="https://mintcdn.com/qoder/aMv0QjdbXL-CIJ29/images/qoder-settings-menu.png?fit=max&auto=format&n=aMv0QjdbXL-CIJ29&q=85&s=34adf7582dcb33d778e970d895d1d4f6" alt="Qoder 设置菜单" width="1048" height="660" data-path="images/qoder-settings-menu.png" />

**2：进入模型面板**

在左侧导航栏中选择「模型」

<img src="https://mintcdn.com/qoder/-C_L-Xnbo1hnztgL/images/qoder-models-panel.png?fit=max&auto=format&n=-C_L-Xnbo1hnztgL&q=85&s=0a607c0f68abe1f330a21b161bd3dde2" alt="模型面板" width="1504" height="848" data-path="images/qoder-models-panel.png" />

**3：添加模型**

点击「+ 添加模型」，选择服务商按需选择所需的模型，并填写您的 API 密钥

<Tip>点击「获取 API 密钥」可跳转至对应服务商的密钥管理页面。</Tip>

<img src="https://mintcdn.com/qoder/aMv0QjdbXL-CIJ29/images/qoder-add-model-dialog.png?fit=max&auto=format&n=aMv0QjdbXL-CIJ29&q=85&s=aeb4b20b3be5a24bf5423243e5defcc7" alt="添加模型对话框" width="1142" height="936" data-path="images/qoder-add-model-dialog.png" />

**4：验证连接**

点击「添加模型」，系统将自动验证连接状态

**5：使用模型**

配置完成后，您可以在对话框中选择配置的模型开始使用

## 管理自定义模型

在「模型」面板中，可对自定义模型执行以下操作：

| **操作** | **说明**         |
| ------ | -------------- |
| 编辑     | 修改模型名称别名       |
| 删除     | 移除模型，删除后不可继续使用 |
| 启用/禁用  | 控制模型是否显示在选择列表中 |

## 常见问题

### 1. 使用自定义模型是否消耗 Credits？

自定义模型的费用由服务商 API 账户直接结算，不占用 Qoder Credits。但 Repo Wiki 使用固定模型，使用时会消耗 Qoder Credits，生成时会有提示。

### 2. 如何获取 API 密钥？

各服务商密钥获取地址：

* **GLM**：[open.bigmodel.cn](https://open.bigmodel.cn/)
* **Kimi**：[platform.moonshot.cn](https://platform.moonshot.cn/)
* **MiniMax**：[platform.minimaxi.com](https://platform.minimaxi.com/)
* **Bailian**：[bailian.console.aliyun.com](https://bailian.console.aliyun.com/)

### 3. 添加模型失败如何处理？

请检查以下事项：

1. API 密钥是否正确，无多余空格
2. API 密钥是否已过期或被禁用
3. 服务商账户余额是否充足
4. 网络连接是否正常

### 4. 是否支持添加同一服务商的多个模型？

支持。可为同一服务商添加不同版本的模型，或使用不同的 API 密钥。

### 5. 使用自定义模型会消耗 Credits 吗？

使用自定义模型本身不消耗 Qoder Credits —— 您直接向服务商支付 API 使用费用。但 Repo Wiki 使用固定模型，无论您的自定义模型配置如何，使用时都会消耗 Qoder Credits，生成时会有提示。
