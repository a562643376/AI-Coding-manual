# 隐私模式

当你使用 TRAE 的服务时，你的聊天交互信息（包括相关代码片段）等信息可能会被用于分析、产品改进和模型训练。

启用隐私模式后，TRAE 不会出于上述目的存储或使用你的任何聊天互动内容（包括相关代码片段）。

## 重要提示

隐私模式不影响你的代码库文件的存储方式，这些文件仍会保存在你的设备本地。

## 启用 “隐私模式”

1. 在 IDE 模式界面中，点击界面右上角的 **设置** 图标，进入设置中心。

- 或

- 在 SOLO 模式界面中，点击对话面板右上角的 **设置** 图标，进入设置中心。

2. 在左侧导航栏中，选择 **账号**，进入 **账号** 面板。

3. 打开 **隐私模式** 开关。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/cb65cdd8ab8d4be5a7d992a268208cea~tplv-goo7wpa0wc-quality:q75.awebp)

- 提示“隐私模式” 仅在登录 TRAE 账户后生效。一旦登出账号，“隐私模式” 便不生效。
