# 企业智能体

智能体（Agent）是面向不同开发场景的编程助手。TRAE 提供内置智能体，同时支持创建自定义智能体 —— 通过灵活配置提示词和工具集，使其更高效地协助你完成复杂开发任务。更多介绍参考智能体概述。

在 TRAE 企业版中，你可创建企业专属智能体。创建后，该智能体将对所有企业成员可见且可用，便于团队协作与能力复用。

## 套餐要求

- 团队版

- 旗舰版

## 创建企业专属智能体

你可以通过智能生成和手动创建两种方式来搭建自定义智能体。

1. 在 AI 对话输入框中输入 **@**，然后点击浮起面板底部的 **创建智能体** 按钮。

- 你将进入以下 **创建智能体** 面板。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/d9901cc5724a42c683f5bd3ba96dad0e~tplv-goo7wpa0wc-quality:q75.awebp)

2. 设置 **类型** 为 **企业专属智能体**。

3. 选择一种方式来创建智能体，包括智能生成和手动创建。详细操作说明参考创建自定义智能体。

4. 点击底部的 **保存** 或 **保存并发布** 按钮。

- 提示企业专属智能体在提交审核后将直接自动通过审核。

## 启用企业专属智能体

创建智能体后，其默认为启用。启用后的智能体将出现在所有企业成员的 IDE 端智能体列表中。

1. 登录 [TRAE 企业版控制台](https://console.enterprise.trae.cn/)。

2. 前往 **企业配置** > **企业智能体**。

- 你将进入 **企业智能体** 列表，展示企业内成员创建的所有智能体。

3. 找到目标智能体，然后打开 **是否启用** 栏处的开关。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/54046f63dc2b409088348042e21abd6a~tplv-goo7wpa0wc-quality:q75.awebp)

## 编辑/删除企业专属智能体

仅智能体创建者可编辑或删除自己创建的智能体。

1. 打开 TRAE IDE。

2. 前往 **设置** > **智能体**。

3. 找到你创建的智能体。

4. 点击其右侧的设置按钮。

5. 在菜单中选择 **编辑** 或 **删除**，然后完成相应操作。
