# 企业文档集

若需将企业内部知识作为 AI 对话的上下文，可构建企业专属文档集。文档集对企业内所有成员可见且可用，助力团队统一知识口径，提升 AI 回答精准度。

## 套餐要求

- 团队版

- 旗舰版

## 存储空间上限

- 团队版：2G

- 旗舰版：4G

## 创建企业内置文档集

1. 登录 [TRAE 企业版控制台](https://console.enterprise.trae.cn/)

2. 前往 **企业配置** > **企业文档集**。

3. 在页面右上角，点击 **+ 新增文档集** 按钮。

- 你将进入 **新建文档集** 面板。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/32fa253fe8db45568e0f577aa335fa59~tplv-goo7wpa0wc-quality:q75.awebp)

4. 输入文档集名称和文档集描述，然后上传文件（支持格式：.md、.txt、.pdf；单个文件大小 ≤ 10MB）。

5. 点击 **保存** 按钮。

- 在 IDE 内的 **内置文档集** 部分，你添加的文档集对所有企业成员可见。

## 编辑/删除企业内置文档集

你可以按需编辑或删除文档集。

1. 登录 [TRAE 企业版控制台](https://console.enterprise.trae.cn/)

2. 前往 **企业配置** > **企业文档集**。

3. 在文档集列表中，找到目标文档集。

4. 点击 **操作** 栏中的 **···** 图标，在菜单中选择所需选项，然后完成相关操作。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/31410dc4f6384b1899e4ff789619464d~tplv-goo7wpa0wc-quality:q75.awebp)

## 在对话中使用文档集

在 AI 对话输入框中，通过 **#Doc** 将文档集作为上下文。
