# 计费项：按量计费 & 席位费

TRAE 企业版包含两种计费项：席位费、按量计费。

## 按量计费

开通按量计费后，当企业内成员资源使用量超过套餐内额度时，若继续调用大模型，将按实际 Token 消耗计费。

### 重要提示

按量计费采用小时后付费方式结算，因此出账存在小时级延时。只有在出账扣款操作完成后，账户才会进入欠费状态；在此之前，企业仍可正常使用 TRAE 企业版，且可能继续产生费用。

### **计费公式**

在大语言模型中，Token 是文本处理的基本单位，可表示为单词、字符或子词片段。

按量计费时，TRAE 内置模型均按 Token 使用量计费，以豆包为例，计费公式如下：豆包模型服务费 = 输入 Token 使用量 × 输入Token 单价 + 输出 Token 使用量 × 输出 Token 单价。

## 席位费

席位为 TRAE 企业版基本售卖单元，不同套餐的席位单价与起购席位量不同。企业可在 TRAE 内添加与席位数量相等的成员；当达到最大可购席位数后，将无法邀请新成员加入企业。

### 席位内权益

席位内包含以下权益：

|  | **功能** | **可用次数** |
| --- | --- | --- |
|  | 问答（Agent / Chat / Builder） | 按 Token 计费。 **提示**：豆包模型的 Token 计费规则为：企业基础版按 1:1 计算；团队版和旗舰版按 1:0.5 计算（相当于五折）。 |
|  | 补全 | 按 Token 计费。 |
|  | InlineChat | 不限 |
|  | Fast Apply | 不限 |
|  | AI Fix | 不限 |

### 席位释放规则

移除成员后，将根据该成员当月在 AI 对话中的 Token 消耗量来决定是否释放席位，规则如下：

- **消耗量 ≤ 100 万**：席位会立即释放。你可邀请新成员加入。

- **消耗量 > 100 万**：席位需等到次月才能释放。届时你才可邀请新成员加入。

### 退订席位

退订席位后，原先占用该席位的成员将无法继续使用 TRAE 企业版。具体的退订规则，参考火山引擎官方的[退订规则说明](https://www.volcengine.com/docs/6269/81013?lang=zh)。

## 欠费关停

席位采用预付费模式，订阅期内若企业账户欠费，TRAE 不会关停权益，成员仍可正常登录使用。

按量计费采用后付费模式，若出账后扣款失败导致企业账户欠费，TRAE 将在欠费 2 小时后自动关停按量计费，届时企业将无法继续使用该服务。待企业充值金额并使账户恢复正常后，方可继续使用按量计费服务。
