# TRAE 企业版快速入门

本文档介绍使用 TRAE 企业版前需要完成的基础准备工作，包括注册火山引擎账号、购买企业版套餐、注册企业账号、邀请团队成员等。

## 支持的操作系统

| **操作系统** | **架构** | **支持的版本/格式** |
| --- | --- | --- |
| macOS | Apple Silicon、Intel | 12.0 或更高版本 |
| Windows | 64 位 (x64) | Windows 10、Windows 11 |
| Linux | 64 位 (x64)、64 位 (ARM64) | `.deb`、`.rpm` |

## 第一步：注册火山引擎账号

前往[火山引擎官网](https://console.volcengine.com/auth/login?redirectURI=%2Ftrae%2Foverview)，注册火山引擎用户账号。

## 第二步：购买企业版套餐

根据实际需求，购买企业版套餐。购买前，建议先阅读以下内容：订阅体系、套餐类型、计费项：按量计费 & 席位费。了解 TRAE 企业版的计费信息后，按以下步骤购买所需的套餐：

1. 前往 [TRAE 企业版首页](https://www.volcengine.com/product/trae)。

2. 点击 **立即下单** 按钮。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/23fb115d10204906bb224777f320a0c9~tplv-goo7wpa0wc-quality:q75.awebp)

- 你将进入企业版套餐购买页。

3. 选择所需的套餐并完成支付。详情参考购买套餐。

## 第三步：注册企业账号

注册 TRAE 企业账号，用于管理企业设置、成员、计费等。该账号将与你所认证的企业关联。

1. 在支付完成后显示的页面中，点击 **前往 TRAE** 按钮。

- 或

- 若你关闭了支付完成后的页面，再次前往火山引擎中的 TRAE 控制台，然后点击 **前往 TRAE** 按钮。

- 你将前往 **注册企业账号** 页面。

- 提示如果你没有进入该页面，说明当前火山引擎账号已购买过 TRAE 企业版，请联系企业的火山引擎管理员进行确认。

2. 填入你的邮箱。建议使用企业邮箱。

3. 勾选 **我已阅读并同意服务条款和隐私协议**。

4. 点击 **立即注册** 按钮。

- TRAE 将向你的邮箱发送验证码。同时，你将进入以下 **验证邮箱** 页面。

5. 前往你的邮箱，获取验证码。

6. 在 **验证邮箱** 页面中，输入验证码，然后点击 **完成验证** 按钮。

- 你将进入 **设置名称与密码**页面。

7. 设置账号的名称和密码，再次确认密码，然后点击 **完成注册** 按钮。

- 注册完成后，你的账号默认成为企业的超级管理员，你将进入以下 TRAE 企业版控制台。

## 第四步：邀请人员

你购买的套餐中包含一定数量的默认席位，因此你可以邀请相应数量的成员加入你的企业。

1. 在 TRAE 企业版控制台中，前往 **企业管理** > **人员管理**。

2. 点击右上角的 **+ 邀请人员** 按钮。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/23af9ee15ab44e3b933954d517498687~tplv-goo7wpa0wc-quality:q75.awebp)

3. 选择邀请方式：邮件邀请、批量上传或链接邀请，然后完成所需操作。详细操作步骤参考邀请人员。

- 提示若套餐内默认的席位数量不足，你可以点击右上方的 **增购席位** 按钮进行增购。详情参考增购席位。

## 第五步：完成企业版配置

根据实际需求，在 TRAE 企业版控制台中配置全局生效的通用设置，包括数据保护、按量付费、内置模型等。详情参考通用设置和模型管理。

![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/153d0e6b511e4985bad2e54886be699f~tplv-goo7wpa0wc-quality:q75.awebp)

## 第六步：安装 TRAE 并完成初始设置

1. 前往 [TRAE 官网](https://www.trae.com.cn/)，点击右上角的 **下载 IDE** 按钮，将安装包下载到本地并完成安装。

2. 启动 TRAE，然后按照界面指引完成初始设置，包括：

- 选择主题和语言。

- 从 VS Code 或 Cursor 导入已有配置。

- 添加 TRAE 相关的命令行。

- 使用已分配席位的邮箱登录 TRAE 企业版。登录时，需选择 **企业用户** 入口。
