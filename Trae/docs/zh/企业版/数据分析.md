# 数据分析

TRAE 企业版提供数据分析面板，展示企业整体的使用数据，包括人员数据、人员活跃度趋势、算法请求统计等。

## 使用限制

- 基础版用户无法使用数据分析功能。

- 团队版用户无法查看 CLI 相关数据。

## 查看数据

你可以点击左侧导航栏中的 **数据分析** 按钮，打开数据分析面板。

1. 登录 [TRAE 企业版控制台](https://console.enterprise.trae.cn/)。

2. 前往 **企业管理** > **数据分析**。

- 你将进入数据分析面板。

- ![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/d7ef83934dc544c4b434dae3408e5348~tplv-goo7wpa0wc-quality:q75.awebp)

3. (可选) 设置筛选项，包括人员和数据周期。

4. (可选) 点击右上角的 **导出洞察报告** 按钮，将数据分析报告导出至本地。

## 数据统计口径

各数据分析面板的统计口径说明如下：

| **看板** | **指标** | **统计口径** |
| --- | --- | --- |
| 人员数据 | 总计使用人员数量 | 使用 IDE 或插件内 AI 能力的用户数量（以账号维度去重）。 |
| 人员数据 | IDE人员数量 | 使用 IDE 内 AI 能力的用户数量（以账号为准）。 |
| 人员数据 | 插件使用人员数量 | 使用插件内 AI 能力的用户数量（以账号为准）。 |
| 人员数据 | CLI 使用人员数量 | 使用 CLI 内 AI 能力的用户数量（以账号为准）。 |
| 人员活跃度趋势 | 总计人员 | 按照筛选时间展示每日所有产品的总使用人数。 |
| 人员活跃度趋势 | IDE 使用人员 | 按照筛选时间展示每日 IDE 的使用人数。 |
| 人员活跃度趋势 | 插件使用人员 | 按照筛选时间展示每日插件的使用人数。 |
| 人员活跃度趋势 | CLI 使用人员 | 按照筛选时间展示每日 CLI 的使用人数。 |
| 核心使用数据 | AI 生成率 | AI 建议被采纳代码/（人工手动编写代码 + AI 建议被采纳代码）x 100%。 |
| 核心使用数据 | AI 生成代码量 | AI 问答和补全累计生成代码量。 |
| 核心使用数据 | CUE 推荐次数 | 补全累计推荐次数。 |
| 对话代码生成趋势 | 生成代码行数 | 按照筛选时间展示对话生成的代码行数。 |
| 对话代码生成趋势 | 采纳代码行数 | 按照筛选时间展示对话生成代码被采纳的行数。 |
| CUE 推荐代码趋势 | 推荐代码次数 | 按照筛选时间展示 CUE 的修改点推荐次数。 |
| CUE 推荐代码趋势 | 采纳代码次数 | 按照筛选时间展示 CUE 推荐的修改点被采纳的次数。 |
| MCP 排行榜 | - | 筛选时间内调用最多的 MCP（以 tool 维度统计调用次数）。 |
| 大模型调用分布图 | - | AI 对话使用的模型分布。 |
| 编程语言分布图 | - | 问答采纳代码的语言分布。 |
