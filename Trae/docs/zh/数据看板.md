# 数据看板

从注册 TRAE 账号起，系统将开始记录你的使用情况。通过数据看板，你可以直观查看每日活跃情况。根据累计活跃天数，系统将自动授予相应的用户称号，以此反映你的使用频率与持续投入。

你还可以在看板中查看更多使用行为数据，包括采纳 TRAE IDE 推荐代码的数量、在不同模式下与 AI 的对话统计、近期的模型调用偏好等。这些数据将有助于你更好地了解自身的使用行为。

登录 [TRAE 官网](https://www.trae.cn/)，点击右上角的头像，然后在菜单中选择 **仪表盘**，查看数据。
