# 多任务并行

多任务管理功能打破传统的串行执行任务的限制，支持在一个项目中同时管理多个任务，提升整体开发效率。

## 使用场景

- **同步进行问答与开发**：你可以在不同任务之间自由切换，例如在进行 Supabase 模块开发的同时，进行辅助问询（如 “Supabase 配置方法”），避免让你和 AI 的思路被限制在一个会话中。

- **并行执行多个开发任务**：你可在同一项目下创建多个独立任务，分别用于不同功能模块的开发（如前端与后端），实现多任务并行推进。

## UI

任务面板位于 SOLO 界面的左侧，展示当前项目中的所有任务及每个任务的状态。

## 相关操作

### **创建新任务**

通过以下方式创建新任务：

- 点击任务面板顶部的 **新任务** 按钮；

- 点击 AI 对话面板右上角的 **新任务** 图标；

- 使用快捷键（macOS：`command + control + N`；Windows：`Ctrl + Alt + N`），对话输入框内会出现输入点光标。

### **重命名/删除任务**

点击任务卡片右上角的 **···**  按钮，在菜单中选择相应的选项并完成所需操作。

### 隐藏/显示任务面板

进入 SOLO 模式后，默认显示任务面板，你可以通过面板右上角的 **隐藏任务** 图标来隐藏任务面板。隐藏后，可通过 **显示任务** 按钮再次打开。
