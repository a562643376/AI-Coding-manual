# 重磅更新：TRAE SOLO 独立端上线

TRAE SOLO 独立端上线。TRAE SOLO 是一款 AI 原生工作台，提供网页版和桌面版，并设有为不同用户群体设计的 More Than Coding (MTC) 与 Code 双模式。TRAE SOLO 的能力在原有 TRAE IDE SOLO 模式的基础上得到了进一步提升，旨在为用户提供统一、高效、可定制的 AI 协作体验，覆盖从专业开发到日常办公的各类场景。
