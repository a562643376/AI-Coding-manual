# 查看 MCP Server 的日志

你可以查看 MCP Server 的运行日志，从而更精确地排查相关问题。

## 入口一：MCP Server 列表

在 MCP Server 列表中，找到目标 MCP Server，点击右侧的齿轮图标，然后在菜单中选择 **日志**。TRAE 会打开 **输出** 面板，展示 MCP Server 相关日志。

![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/cca6c2a1483b4ac69c1f2c98e70363a2~tplv-goo7wpa0wc-quality:q75.awebp)

若 MCP Server 运行出错，可以将鼠标悬浮在对应的 MCP Server 上，然后在弹出的错误信息面板中点击 **日志**。

![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/51915050980446b5a86202ef93c150b8~tplv-goo7wpa0wc-quality:q75.awebp)

## 入口二：输出面板

1. 使用快捷键（macOS：Command + Shift + U；Windows：Ctrl + Shift + U）打开 **输出** 面板。

2. 在右上角的下拉菜单中，选择 **MCP Server Host**。

- 面板中将展示 MCP Server 相关日志。

![image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/185b12c781e2485ebb6622c680bbcb4b~tplv-goo7wpa0wc-quality:q75.awebp)
